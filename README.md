# 📦 Supply Chain Sales Prediction (Machine Learning Project)

 A machine learning project focused on predicting sales using supply chain data with high accuracy.


## Project Overview

This project focuses on building a machine learning model to predict sales using a supply chain dataset. The goal is to analyze key factors affecting sales and develop a predictive model with strong performance.

---

## Objective




* Predict sales based on historical supply chain data
* Identify important features influencing sales
* Build and evaluate a regression model

---

## Dataset

* **Source**: DataCo Supply Chain Dataset
* Contains information about:

  * Orders
  * Products
  * Customers
  * Shipping details

---

##  Workflow

### 1. Data Preprocessing

* Handled missing values
* Removed duplicates
* Cleaned inconsistent data
* Encoded categorical variables

### 2. Exploratory Data Analysis (EDA)

* Analyzed feature distributions
* Identified correlations
* Visualized key trends

### 3. Feature Engineering

* Selected important features
* Created meaningful input variables

### 4. Model Building

* Applied **Linear Regression**
* Split data into training and testing sets

### 5. Model Evaluation

* Evaluated using R² Score
* Achieved strong performance on test datata

---

## Results

- Model Used: Linear Regression  
- R² Score: 0.95  
- Strong predictive performance on test data  

## Key Insights

- Certain features strongly influence sales  
- Customer and order patterns affect predictions  
- Feature engineering improved model performance
